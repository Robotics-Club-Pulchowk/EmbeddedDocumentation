Getting Started
===============

Here is a list we commonly use.

- Controllers:
    - STM32 Controllers
        - STM32F407VG (Discovery Board)
        - STM32F103C8 (Blue Pill)
        - STM32F103C6 (Black Pill commonly found in Nepal)
    - ESP Controllers
        - ESP32 (ESP32 WROOM32)
        - ESP8266 (ESP8266 NodeMCU)
    - Raspberry Pi Controllers
        - Raspberry Pi Pico
        - Raspberry Pi Pico W
    - Arduino Controllers
        - Arduino Uno
        - Arduino Nano
        - Arduino Mega

- Debuggers:
    - ST-Link V2
    - JLink EDU Mini


- IDEs and SDKs:
    - Arduino
    - STM32Cube Toolchain
    - Pico SDK
    - ESP-IDF


- Scripting and Programming Laguages:
    - Bash
    - Make
    - C
    - C++
    - Python
